# homebrew-queueski

Homebrew tap for Queueski.

After the first Queueski release is published from the monorepo, install the CLI
with:

```bash
brew install lookevink/queueski/qsk
```

This repository is release infrastructure only. Source code lives in
[`lookevink/queueski`](https://github.com/lookevink/queueski), and the `qsk`
formula is generated from that repository's `dist` release configuration.
